# 新闻参考

当用户询问「最新新闻」、「今天有什么新闻」或「X类别的新闻」时，使用以下类别和网址的 **web_fetch** 工具获取内容，然后从返回的内容中提取标题和要点并回复用户。

## 类别和来源

| 类别      | 来源                    | URL |
|---------------|---------------------------|-----|
| **政治**  | 人民日报 · 中国共产党新闻 | https://cpc.people.com.cn/ |
| **财经**   | 中国经济网        | http://www.ce.cn/ |
| **社会**   | 中国新闻网 · 社会      | https://www.chinanews.com/society/ |
| **国际**     | 中国国际电视台                      | https://www.cgtn.com/ |
| **科技**      | 科技日报 | https://www.stdaily.com/ |
| **体育**    | 央视体育               | https://sports.cctv.com/ |
| **娱乐** | 新浪娱乐   | https://ent.sina.com.cn/ |

## 使用方法 (web_fetch)

1. **明确用户需求**：确定哪个或哪些类别（政治/财经/社会/国际/科技/体育/娱乐），或选择1-2个类别获取新闻。
2. **选择网址**：使用该类别的表格中的网址；可以同时获取多个类别。
3. **调用工具**：调用 **web_fetch** 工具，传入包含所选网址的列表：
   ```json
   {"urls": ["https://www.chinanews.com/society/", "https://sports.cctv.com/"]}
   ```
   替换为您需要获取的网址列表。
4. **处理结果**：从返回的结果中提取每个网址的标题、描述和内容，重点关注新闻标题和摘要。
5. **总结回复**：按时间或重要性组织简短列表（标题 + 一两句内容 + 来源）；如果网站无法访问或内容为空，请告知并建议其他来源。

## 注意事项

- 网站更新时页面结构可能会改变；如果提取失败，请告知并建议用户直接打开链接。
- 可以一次性获取多个类别的新闻，无需为每个网址单独调用工具。
- 您可以在回复中包含原始链接，以便用户可以打开它。
